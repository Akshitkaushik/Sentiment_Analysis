### Steps for AFREALLOC Rule Not on Plan:

1. RPA will receive the Input report from ---- [TBD].
2. Login to EASY and select the database as given in the input report.
3. Enter code "CHGR" and press [Enter].
4. Insert Plan Number -> Press [Enter], then press F12.
5. Check if a pop-up exists, then click on the OK button.
6. Select "Change Group Account Recovery Information" -> Press [Enter].
7. Since the rule is not on plan:
    - Search Rule ID row by row in the RULE ID column.
    - If Rule ID is not found, insert a new row using Ctrl+F12 or the down arrow key.
    - Check if a pop-up appears, then click the OK button.
    - Insert the Rule ID from the spreadsheet in the [RULE ID column] and press [Enter].
    - Select the rule ID from the below screen, then click OK.
8. Tab over to the effective date and insert the effective date from the spreadsheet in the [EFT DATE (if new rule added) column].
9. Press Tab 4 times to reach the Statement Label column and insert the STATEMENT LABEL from the spreadsheet in the [STATEMENT LABEL column].
10. Press F6 to save.
11. Press F12 once to bring up the next screen.
12. Press F12 to go to the description screen, then insert the DESCRIPTION from the spreadsheet in the [DESCRIPTION column].
13. Press F12 twice.
14. Insert the plan number, then press [Enter] 6 times to insert the FUND TYPE from the spreadsheet [CALC = Y column].
    - Split the value with a comma, then enter the first value, press [Enter], and insert Y.
15. Press the down arrow key.
    - Insert the plan number -> Press [Enter] 6 times to insert the second FUND TYPE from the spreadsheet.
    - Press [Enter], insert Y, then repeat the process for the third value.
16. If a pop-up appears, click the OK button.
17. Press F12 to reach the % of BASIS.
18. Insert 0 -> Press [Enter], then press F6 to save.
19. If a pop-up appears, click the OK button.
20. Press F11 four times to return to the previous screen.
21. Press the down arrow key to go to "Maintain Debit Method Order Rules" and press F12 three times to land here.
22. Insert the plan number -> Press [Enter], then insert the sequence number (start with 1 and increment by 1 each time).
23. Press [Enter] 6 times to reach the FUND TYPE column.
24. Insert STK from the spreadsheet [DMO = Y column, split the value with a comma, then enter the first value].
25. Press [Enter], then insert Y.
26. Press the down arrow key, and if a pop-up appears, press [Enter] to go through it.
27. Repeat steps 23-26 for the second and third STK values.
28. Press F6 to save, then press Ctrl+F9.
29. Insert the Begin Date, End Date, and Schedule Date from the spreadsheet [BEGIN, END, SCHE Date columns].
    - Ensure the schedule date ends with "00:00".
30. Hit [Enter] to move to the REQUESTED AMOUNT (the system should insert a date into the EFFECTIVE column).
31. Insert the amount from the spreadsheet [Requested Amount column].
32. Press F6 to save -> Press F3 twice to return to CHGR.
33. Continue to the next record.
34. Note: If any column value is already filled in, the bot will overwrite that value.

### Steps for AFREALLOC Rule on Plan:

1. RPA will receive the Input report from ---- [TBD].
2. Login to EASY and select the database as given in the input report.
3. Enter code "CHGR" and press [Enter].
4. Insert Plan Number -> Press [Enter], then press F12.
5. Check if a pop-up exists, then click the OK button.
6. Select "Change Group Account Recovery Information" -> Press [Enter].
7. Since the rule is on the plan:
    - If the rule appears twice or more, DO NOT use a rule with a term date.
8. Press F12 to bring up the selection screen.
9. Press F12 to go into "Maintain Combo Details."
10. Ensure the description is displayed as expected.
11. Press F12 twice to check for the standard three exclusions.
    - If not present, they need to be added.
12. Press F12 once to reach the % of BASIS section.
13. Use the down arrow key until a clean row with no data appears.
14. Insert 0 and press F6 to save.
15. Press F11 four times to return to the previous screen.
16. Use the down arrow key to reach "Maintain Debit Method Order Rules" and press F12.
17. Press F12 to go to the data insert columns.
    - Ensure the three standard entries are there; if not, add them.
18. Press Ctrl+F9 to go to the assessment row, then insert a new row using Ctrl+F12.
19. Insert the Begin Date, End Date, and Schedule Date from the spreadsheet [BEGIN, END, SCHE Date columns].
    - Ensure the schedule date ends with "00:00".
20. Hit [Enter] again to move to the REQUESTED AMOUNT.
    - The system should insert a date into the EFFECTIVE column.
21. Insert the amount from the spreadsheet.
22. Press F6 to save -> Press F3 twice to return to CHGR.
23. Continue to the next record.
24. We then click OK and press F3 twice to exit the plan and return to CHGR to begin the next plan in the list.
